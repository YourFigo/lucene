# `lucene全文检索基本使用`
- 手打亲测可用 :smile:
---
## `目录如下`
- cn.figo.lucene.IndexManager：创建索引库、查询索引
- cn.figo.lucene.LuceneFirst：索引库的维护
- cn.figo.lucene.SearchIndex：范围查询和QueryParser进行查询
- hotword.dic：IK分析器的扩展关键词列表
- IKAnalyzer.cfg.xml：IK分词器的扩展配置文件
- stopword.dic：IK分词器的停用词列表
